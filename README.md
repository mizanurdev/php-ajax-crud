## Purpose
The purpose of this project is to serve as a practice exercise for enhancing your skills in PHP and AJAX by building a simple, dynamic CRUD application.

## Database Information
- **Database Name:** php_ajax_crud
- **Note:** You can create the necessary tables by running `database/migration.php` or import the provided SQL file (`php_ajax_crud.sql`).


## Getting Started
1. Clone the repository or download the project files.
2. Set up a local server environment (e.g., XAMPP, WAMP).
3. Import `php_ajax_crud.sql` into your MySQL database or execute `database/migration.php` to create the required tables.
4. Configure `database/config.php` with your database credentials.
5. Open `index.php` in your browser to start using the application.

## Features
- **AJAX Integration:** Asynchronous data loading and updating using AJAX.
- **Create, Read, Update, Delete (CRUD) Operations:** Manage student records without refreshing the page.
- **Modular Layout:** Includes separate header and footer files for better code organization.

## Dependencies
